The client

A volunteer program that feeds a county.

Harvest Pierce County is a program of the Pierce Conservation District. Its Gleaning Project harvests surplus produce from local farms and backyard trees and gives it to food banks, shelters, and families. The program has saved 499,333 pounds of food since 2014 and runs across more than 90 community food projects. The website was supposed to bring volunteers in. Instead it got in the way.

The problem

The old site needed a developer for everything.

A site audit surfaced six issues. The root of most of them was simple: staff could not change a word without help.

Legacy technical debt

A PHP and HTML backend pulled from an undocumented database. Any edit meant calling the database owner or a developer.

Staff dependency

The program manager could not update content alone. Routine changes waited on someone else.

Disjointed navigation

All Gleaning content sat on one long scrolling page, mixed with Pierce Conservation District material.

Broken user flows

Volunteer sign-up jumped to a separate external platform, breaking the visual and navigational flow.

No About page

First-time visitors had no clear introduction to the Gleaning Program or its mission.

Parent-site confusion

Links kept sending users back to the parent district site, so people lost track of where they were.

The choice

We picked the project on evidence, not gut.

Three community projects were on the table. I built a decision matrix that scored each one on seven weighted criteria. Harvest Pierce County won by a clear margin.

The build

Three hurdles, three calls.

Legacy database

Audit, then split

The team worked with the database owner to audit each page and decide what needed the old backend versus a clean rebuild.

Shared editing

Lock the lanes

Hosting limited simultaneous logins, so we set page ownership and daily handoffs to stop overwrites.

Messy legacy files

Rebuild clean

Rather than migrate years of tangled files, the team built a fresh WordPress instance and moved only approved content.

The outcome

Four clear pages, and a site the client owns.

The team delivered a homepage, a Get Involved hub, an About page that never existed before, and a Meet the Team page. Most of all, the program manager can now edit it without a developer.
